数据类型是程序的基础：它告诉我们数据的意义以及我们能在数据上执行的操作。

基本数据类型

C++ 定义了一套包括 算数类型 和 空类型 在内点基本数据类型。其中 算术类型 包括 字符、整数型、布尔值和 浮点数。空类型不对应具体的值，仅用于一些特殊点场合，例如最常见点是，当函数不返回任何值时使用空类型作为返回类型。
